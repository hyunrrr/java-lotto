## 기능 목록

1. 로또 구입 기능 - LottoGame
2. 로또 번호 생성 기능 - Lotto
3. 당첨 번호 입력 기능 - LottoGame
4. 로또번호 당첨번호와 비교하는 기능 - Lotto
5. 맞춘 개수를 로또 등수로 바꿔주는 기능 - LottoRewardLogic
6. 적절한 로또번호 생성기 만드는 기능 - LottoGame
    - 수동로또 개수 한도 내인지 확인
    - 수동로또번호 숫자 6자리인지 확인
    - 수동로또번호 1-45사이인지 확인
