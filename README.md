- [온라인 코드 리뷰 과정](https://github.com/woowacourse/woowacourse-docs/blob/master/maincourse/README.md)

## 기능 목록
### LottoNumber
1. 유효한 로또번호 범위내 숫자인지 확인

### Lotto
1. 번호 생성
2. 맞은숫자 count
3. 보너스번호 포함 확인

### LottoGames
1. 로또 구입
   - 수동 로또 구입 개수 유효한지 확인
2. 정답 입력
3. 당첨 통계
4. 수익률 계산

### AutoLottoGenerator
1. 랜덤 번호 6개 생성
2. 번호 전달

### ManualLottoGenerator
1. 번호 전달
2. 번호 6개인지 확인
3. 중복 번호 없는지 확인

### WinningLottoGenerator
1. 번호 전달
2. 번호 6개인지 확인
3. 번호6개 + 보너스번호 중복 없는지 확인
